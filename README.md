# ONVIF Kamera-Überwachungssystem

Automatisches Kamera-Überwachungssystem für ONVIF-kompatible Kameras mit Live-Stream, Aufnahme und Web-Dashboard.

## Schnellstart

### Windows
Doppelklicken Sie auf `start.bat`

### macOS/Linux
Doppelklicken Sie auf `start.sh` oder führen Sie im Terminal aus:
```bash
./start.sh
```

## Vollständige Anleitung

Siehe **ANLEITUNG.md** für eine detaillierte Installations- und Bedienungsanleitung.

## Features

- ✅ Automatische Kamera-Erkennung im Netzwerk
- ✅ Live-Stream-Ansicht im Web-Dashboard
- ✅ Automatische Video-Aufnahme mit Audio (wenn FFmpeg installiert)
- ✅ Konfigurierbare Aufnahme-Einstellungen
- ✅ Automatische Bereinigung alter Aufnahmen (24h)
- ✅ Einfache Bedienung über Web-Interface

## Systemanforderungen

- Python 3.9+
- FFmpeg (optional, für Audio empfohlen)
- Windows 10/11 oder macOS
